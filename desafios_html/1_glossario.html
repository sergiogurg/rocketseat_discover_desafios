Glossário de tags HTML

Tags <h1> até <h6>
Servem para construir títulos e subtítulos. Existe uma hierarquia entre eles, na qual o <h1> é o título principal, e o <h6> é a subdivisão de menor hierarquia.
Tag <p>
Serve para colocar o texto em parágrafos.
Tag <img>
É responsável por apresentar imagens no código HTML. Seus atributos principais são href, a qual recebe o endereço de onde está localizada a imagem desejada, e alt, que exibe um texto alternativo no caso de a imagem falhar em ser renderizada.
